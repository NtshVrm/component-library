INPUT
Inputs are used to accept various types of information from a user.
Infinity UI offers a basic input component with three types of validation color schemes : SUCCESS | WARNING | ERROR.

  <div class="input">
    <input class="input-box basic" type="text" />
    <label class="label-input basic-text">Email</label>
  </div>

</div>
