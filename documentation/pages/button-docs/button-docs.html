BUTTON
Buttons are the simplest form of user interaction and act as indicators to a user about necessary actions.
Infinity UI offers four types of buttons: FILLED | OUTLINE | ICON | FLOATING ACTION in
PRIMARY | SECONDARY | ERROR color themes.
FILLED BUTTON

<div class="button-container">

  <!-- PRIMARY FILLED BUTTON -->
  <button class="button filled primary-filled">Primary</button>

  <!-- SECONDARY FILLED BUTTON -->
  <button class="button filled secondary-filled">Secondary</button>
  
  <!-- ERROR FILLED BUTTON -->
  <button class="button filled error-filled">Error</button>

</div>

<div class="button-container">
